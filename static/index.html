local media inspector

minfo

Generate MediaInfo or BDInfo, then download four screenshots in one click.

Tooling

MediaInfo CLI, BDInfo CLI, FFmpeg + FFprobe must be on PATH or set via env.

For large inputs, prefer a server path.

Start typing to autocomplete under MEDIA_ROOT (default: /media).
